# Duwane's Landscaping and Lawn Service Website

This is a modern, responsive website for Duwane's Landscaping and Lawn Service, built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- **5 Pages**: Home, Services, About, Gallery, Contact
- **Responsive Design**: Optimized for mobile, tablet, and desktop
- **Fast Loading**: Built with Next.js App Router and optimized images
- **Animations**: Smooth entrance animations using Framer Motion
- **Contact Form**: Functional contact form UI (needs backend integration)
- **Click-to-Call**: Prominent phone numbers for mobile users

## Client-friendly feature diagram

See `docs/FEATURES.md` for a page map + feature diagram you can share with clients.

## Tech Stack

- [Next.js](https://nextjs.org) - React Framework
- [Tailwind CSS](https://tailwindcss.com) - Styling
- [Framer Motion](https://www.framer.com/motion/) - Animations
- [Lucide React](https://lucide.dev) - Icons

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Customization

- **Images**: The project uses placeholder images from Unsplash. Replace them with your own images in the `public` folder or update the URLs.
- **Contact Form**: The contact form currently logs data to the console. Integrate with a service like Formspree or EmailJS for actual email functionality.
- **Content**: Update the text in the page files (`app/**/*.tsx`) to match your business details.
